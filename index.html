PROJECT 01 · FLAGSHIP

Hydraulic Manipulator Digital Twin

A real-time twin of a Parker cylinder bench: a 4-state Python plant model validated against MATLAB/Simulink, with an Extended Kalman Filter running on a Raspberry Pi, streaming over TCP/IP + Firebase from DAQ hardware into a live Flask dashboard. Friction fully characterized (Stribeck/LuGre).

Outcome: end-to-end twin closing the loop from physics → estimation → edge → web, used as the core of my doctoral platform.

PROJECT 03

Real-Time Physics — XPBD in Unity

Real-time soft-body and deformable-body simulation in Unity built on XPBD (Extended Position-Based Dynamics) with energy-derived, neo-Hookean constraints — paired with an OpenGL course (rendering pipeline up, not engine-down).

Outcome: the real-time, interactive counterpart to my offline CFD/SPH work — fast, stable physics a twin can actually run live.

04 / history

Resume

The formal record behind the projects above — full version available as a PDF.

Engineer and PhD candidate specializing in physics-based computational modeling and digital twins, with equal strength in simulation and hands-on experimentation. Builds first-principles and data-driven models — multiphysics/CFD, dynamic system simulation, discrete-event simulation, real-time state estimation — and validates them against experimental and operating data. Industry and applied work span a manufacturing-cell digital twin at Hitachi and discrete-manufacturing systems in Purdue's Smart Learning Factory.
